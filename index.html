Project

🛠️
AI 기반 2D 이미지의 3D 굿즈 제조 플랫폼

2026 졸업작품 · 백엔드 & 서버 담당

⬡ GitHub →

2D 이미지 생성부터 3D 모델 자동 변환, 파트너 업체 견적·결제까지 하나의 흐름으로 처리하는 웹 기반 굿즈 제작 의뢰 플랫폼. 사용자가 캐릭터 이미지를 업로드하면 AI가 자동으로 3D 모델을 생성하고, 제조 업체에 견적 요청과 결제까지 이어진다.

  • Spring Boot 기반 RESTful API 설계 및 구현. Controller – Service – Repository 레이어 분리와 전역 예외 처리로 일관된 응답 구조 확보
  • Access Token / Refresh Token 이중 인증 구현. Refresh Token은 HttpOnly Cookie로 관리하여 XSS 공격 차단, 이메일 인증 가입 흐름 구현
  • MySQL로 회원·주문·결제·에셋·커미션 등 핵심 도메인 테이블 설계. Redis로 토큰 블랙리스트 및 세션 캐싱 처리
  • AI 처리 서버(ComfyUI)와 비동기 연동. 3D 모델 생성 상태를 폴링 방식으로 추적하고 완료 시 실시간 알림 발송
  • AWS S3 파일 업로드, CDN 연동, 외부 결제 API, 메일 인증 서비스 통합
  • JUnit5 · Mockito · MockWebServer로 단위/통합 테스트 작성. JaCoCo 커버리지 측정, SonarCloud 코드 품질 관리
  • Docker 컨테이너화 및 GitHub Actions 기반 CI/CD 파이프라인 구성, AWS EC2 배포
